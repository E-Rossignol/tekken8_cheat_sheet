# Tekken Cheat Sheet deliverables

## Overview

On this folder you can access the finished product.
1. You can install it using the 'installers/tekken_cheat_sheet_installer.exe' file. Once the installation is complete, you can run it on your Windows machine.
2. You can also directly run the app using the 'raw deliverables/tekken_cheat_sheet.exe' file.
3. You can find screenshots of the app in the '~/deliverables/screenshots/' folder.
4. You can find a demonstration video at the following link: [https://youtu.be/8CjFosyT-rE]
