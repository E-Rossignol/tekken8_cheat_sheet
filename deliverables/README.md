# Tekken Cheat Sheet deliverables

## Overview

On this folder you can access the finished app.
1. You can install it using the 'installers/TekkenCheatSheet.exe' file. Once the installation is complete, you car run it on your Windows machine.
2. You can also directly run the app using the 'raw deliverables/tekken_cheat_sheet.exe' file.
3. You can find screenshots of the app in the './screenshots/' folder.
